# Purr Clicker

Purr Clicker is a browser-based incremental game where users generate rewards through clicking and progression mechanics. Players accumulate points over time and unlock paw badges as milestones, creating a simple but engaging gameplay loop.

## Overview

The game is built around core incremental design principles: user interaction drives early progress, while scaling mechanics allow for faster growth over time. The badge system introduces progression goals and encourages continued play.

## Features

- Click-based reward system  
- Progressive scaling of resource generation  
- Paw badge collection tied to milestones  
- Responsive user interface  
- Lightweight browser performance  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript

## How to Play

1. Click the main button to generate rewards  
2. Accumulate points  
3. Unlock paw badges at milestones  
4. Continue progressing to increase output
